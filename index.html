Introduction to electronics with Arduino

Arduino assignement:
Using Tinkercad Circuit, create a "traffic light trigger" based on circuits and code from "Hello Arduino", "Blinking Leds" and "Using Button".

Traffic Light Trigger:
Create a circuit with 3 LEDs (1 Red, 1 Orange, 1 Green) and one button. On boot, Red LED should be turned on, Orange and Green ones should be turned off. When the button is pressed, Red turns off and Green turns on for a few seconds; then Green turns off and Orange turns on for 1 second; finally Orange turns off and Red turns on, until the button is pressed again.
